## 실행 방법

1. `npm i` // 인스톨

2. `npm start` // app.js 실행됨.

## 데이터베이스 마이그레이션

1. `npx sequelize db:migrate`

2. `npx sequelize-cli db:seed:all`

3. `npm start`
